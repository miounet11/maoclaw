# Support Scope

This page is the English support-scope summary for `Maozhua v0.1`.

## Primary v0.1 Support

- macOS
- interactive CLI
- print and JSON output
- baseline RPC
- Anthropic, OpenAI, Gemini, and Azure OpenAI
- `read`, `write`, `edit`, `bash`, `grep`, `find`, `ls`
- save, continue, resume, fork, and compact session flows

## What v0.1 Is Intended To Guarantee

- clear install and first-run path
- baseline provider configuration
- usable main-path tooling behavior
- workable code understanding and lightweight edit flows

## What v0.1 Does Not Promise

- coverage of every historical toolchain and external ecosystem
- equal maturity across every platform
- fully stable behavior across every advanced integration case
