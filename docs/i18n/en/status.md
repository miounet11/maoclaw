# Status

This page is the English public status summary for `猫爪 / maoclaw`.

## Proven Today

- The CLI is installable and usable for terminal-first AI coding workflows.
- Interactive mode, one-shot output, baseline RPC, and session save/resume flows are available.
- The current release is already suitable for real local development workflows.
- The repository already provides public docs, install scripts, and macOS packaging paths.

## Public Release Identity

- Product name: `猫爪`
- Repository and package name: `maoclaw`
- Runtime command: `pi`
- Official website: `https://xinxiang.xin`
- Public documentation languages: Chinese, English, and Japanese

## Current Boundaries

- This is a focused trial release, not a broad enterprise-stable release.
- Deeper RPC, SDK, and external ecosystem stability is still improving.
- macOS is the primary release platform today.
