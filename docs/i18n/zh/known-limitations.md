# 已知限制

本页说明 `猫爪 v0.1` 当前明确存在的边界。

## 主要限制

- 这是聚焦试用版，不是全场景默认稳定版
- 更深的 SDK / RPC 场景仍可能需要适配
- 第三方插件生态不承诺直接兼容
- Linux / Windows 不是当前首要发布平台
- 企业代理、离线、复杂安装环境可能需要额外处理

## 命名说明

当前对外产品名是 `猫爪 / maoclaw`，运行命令仍是 `pi`。  
这会带来一定认知成本，因此文档需要明确解释。

## 使用建议

- 首轮试用优先从 macOS + 一个 Provider + CLI 开始
- 先验证核心问题分析和代码理解工作流
- 再逐步验证 RPC、扩展和更复杂场景
