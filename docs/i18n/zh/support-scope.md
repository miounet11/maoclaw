# 支持范围

本页是 `猫爪 v0.1` 的中文支持范围摘要。

## 首版重点支持

- macOS
- 交互式 CLI
- print / JSON 输出
- 基础 RPC
- Anthropic、OpenAI、Gemini、Azure OpenAI
- `read`、`write`、`edit`、`bash`、`grep`、`find`、`ls`
- 会话保存、继续、恢复、分支、压缩

## 首版重点保证

- 安装和首次启动路径清晰
- Provider 基础配置可完成
- 核心工具主路径可用
- 常见代码理解与轻量修改场景可跑通

## 当前不承诺

- 覆盖所有历史工具链与外部生态
- 所有平台都达到同等成熟度
- 所有高级集成场景都已经稳定
