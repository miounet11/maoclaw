# 版本说明

`猫爪 v0.1` 是首个公开试用版本。

## 本版重点

- 终端优先 AI coding workflow
- 安装、配置、会话、基础集成的真实可用路径
- 多语言 GitHub 文档体系
- 面向 macOS 的桌面分发能力

## 适合谁

- 想试用 AI coding agent 的开发者
- 想评估本地优先工作流产品的团队
- 想通过 RPC 或桌面壳层接入运行时的集成方

## 当前边界

- 不应过度承诺所有场景已经稳定
- 不应对外宣称所有第三方生态都可直接复用
- 不应把内部工程文档当成用户文档
