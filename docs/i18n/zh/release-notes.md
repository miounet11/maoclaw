# 版本说明

当前公开发布线有两个用户必须先认清的节点：

- 2026-03-30 的 `v0.1.13`：当前最新正式公开发布
- 2026-03-28 的 `v0.1.12`：上一正式公开发布
- 2026-03-16 的 `v0.1.11`：较早正式公开发布
- 2026-03-15 的 `v0.1.9`：`maoclaw` 首个官方开源发布

## 当前发布线重点

- 同一运行时下的终端优先 AI coding workflow
- 真实可用的安装、配置、会话和集成路径
- 面向多 Agent 本地工作流的 terminal mesh 协同
- 基于同一运行时的原生桌面安全态势与 readiness 可见性
- macOS 桌面分发加上 Windows 可携式桌面包
- 明确发布边界，而不是夸大兼容面

## 当前构建升级重点

- 继续打磨 onboarding 与 zero-config 自动化路径
- 持续补强 agent 管理、runtime 治理与 mesh orchestration 表面
- 在当前 Windows 可携式桌面包之外继续完善分发体验

这些构建级升级是真实的，但除非进入 `CHANGELOG.md`，否则不应对外说成独立正式发布。
