# 当前状态

本页是 `猫爪 / maoclaw` 的中文公开状态摘要。

## 已验证

- CLI 已可安装并可用于终端优先的 AI coding workflow
- 交互模式、单次输出、基础 RPC、会话保存与恢复已经可用
- 当前版本已经可以支撑真实的本地开发工作流
- 仓库中已经提供公开文档、安装脚本和 macOS 打包路径

## 当前公开发布内容

- 产品名：`猫爪`
- 仓库与发布名：`maoclaw`
- 运行命令：`pi`
- 官方网站：`https://xinxiang.xin`
- 对外文档语言：中文、英文、日文

## 当前边界

- 当前版本是聚焦试用版，不是广泛企业稳定版
- 更深的 RPC / SDK / 外部生态稳定性仍在持续完善
- macOS 是当前首要发布平台
