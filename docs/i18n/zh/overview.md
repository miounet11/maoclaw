# 开源概览

`猫爪 / maoclaw` 是一个本地优先、终端优先的 AI coding agent。

它的公开定位是：

- 来自中国的原创产品
- 面向开发者的实际工作流工具
- 可用于 CLI、自动化输出、RPC 接入和 macOS 桌面分发

## 当前公开能力

- 交互式终端使用
- 单次问题与结构化输出
- 内置文件、搜索、shell 工具
- 会话持久化与恢复
- Provider 配置
- skills、prompts、themes、基础扩展运行时

## 为什么这样发布

公开仓库应该优先承载：

- 运行时代码
- 安装与打包脚本
- 用户文档
- 安全文档
- Provider 与集成说明
- 版本记录

而不应该把内部策略、临时规划和商业操作文档混成默认入口。
