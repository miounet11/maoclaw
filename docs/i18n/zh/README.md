## 产品定位

- 中文名称：`猫爪`
- 英文 / 全球名称：`maoclaw`
- 日文名称：`猫爪 / maoclaw`
- 当前兼容命令：`pi`
- 当前公开定位：独立产品、迁移友好、终端优先、可扩展

## 当前最适合的使用方式

- 本地终端交互式使用
- 单次分析 / 自动化输出
- 通过 `pi --mode rpc` 接入 IDE 或外部客户端
- 在 macOS 上使用桌面壳层进行图形化操作
