# 既知の制約

このページでは `猫爪 v0.1` の明確な境界を要約します。

## 主な制約

- これは焦点を絞った試用版であり、あらゆる場面での既定安定版ではありません
- より深い SDK / RPC 利用では追加適応が必要な場合があります
- サードパーティ製プラグインの各エコシステムが無調整で動くことは約束しません
- Linux と Windows は現時点で主力公開プラットフォームではありません
- 企業プロキシ、オフライン、複雑な導入環境では追加対応が必要になる場合があります

## 命名に関する補足

公開上の製品名は `猫爪 / maoclaw`、実行コマンドは `pi` です。  
この差は理解コストを生むため、ドキュメントで明確に説明する必要があります。

## 推奨アプローチ

- まず macOS、1 つの Provider、CLI から始める
- 最初にコード理解と分析フローを確認する
- その後で RPC、拡張、より複雑なワークフローへ広げる
