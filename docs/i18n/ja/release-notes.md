# リリースノート

`猫爪 v0.1` は最初の公開試用版です。

## このリリースの焦点

- ターミナル中心の AI コーディングワークフロー
- 実用的なインストール、設定、セッション、基本連携経路
- 多言語 GitHub ドキュメント体系
- macOS 向けデスクトップ配布

## 想定ユーザー

- AI コーディングエージェントを評価したい開発者
- ローカルファーストなワークフローツールを検討するチーム
- RPC やデスクトップシェル上で連携したいインテグレータ

## 現在の境界

- すべての場面で安定していると過度に主張しない
- すべての外部エコシステムの直接再利用を約束しない
- 内部向け技術文書をそのまま利用者向け文書にしない
