# 現在の状態

このページは `猫爪 / maoclaw` の日本語版公開ステータス要約です。

## 現時点で確認できていること

- CLI はインストール可能で、ターミナル中心の AI コーディングワークフローに利用できます。
- 対話モード、単発出力、基本 RPC、セッション保存と再開が利用できます。
- 現在のリリースでも、実際のローカル開発ワークフローに使える状態です。
- リポジトリには公開ドキュメント、インストールスクリプト、macOS パッケージング手順があります。

## 公開リリースの識別

- 製品名: `猫爪`
- リポジトリ / パッケージ名: `maoclaw`
- 実行コマンド: `pi`
- 公式サイト: `https://xinxiang.xin`
- 公開ドキュメント言語: 中国語、英語、日本語

## 現在の境界

- これは焦点を絞った試用版であり、広範囲な企業向け安定版ではありません。
- より深い RPC、SDK、外部エコシステムの安定性は引き続き改善中です。
- 現時点の主力公開プラットフォームは macOS です。
