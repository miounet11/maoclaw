# オープンソース概要

`猫爪 / maoclaw` は、ローカルファーストかつターミナルファーストな AI エージェントフレームワークです。

公開上の位置づけは次の通りです。

- 中国で独自開発された製品
- エージェント開発と実務ワークフローのための製品
- CLI、構造化出力、RPC 連携、macOS デスクトップ配布に対応

## 公開されている主な機能

- 対話型ターミナル利用
- 単発プロンプトと構造化出力
- ファイル、検索、shell の内蔵ツール
- セッション保存と再開
- Provider 設定
- skills、prompts、themes、基本拡張ランタイム

## フレームワークとしての価値

`maoclaw` は直接利用だけでなく、次のような構築にも向いています。

- 社内 AI エージェントシステム
- 業務特化型エージェントワークフロー
- 単一ランタイム上の製品化クライアント

## 言語技術とランタイムの強み

- Rust 2024 ベースのランタイムと配布モデル
- 起動とリソース使用の低オーバーヘッド
- セッション、ツール、Provider、連携面を明示的に制御
- CLI、自動化、RPC、デスクトップ配布を一つの基盤で支える

## なぜこの公開構成なのか

公開リポジトリは主に次を担うべきです。

- ランタイムコード
- インストールとパッケージング用スクリプト
- ユーザー向けドキュメント
- セキュリティ文書
- Provider と連携のガイド
- リリース履歴

内部向け戦略メモや一時的な計画資料を、製品の標準入口にすべきではありません。
