# オープンソース概要

`猫爪 / maoclaw` は、ローカルファーストかつターミナルファーストな AI コーディングエージェントです。

公開上の位置づけは次の通りです。

- 中国で独自開発された製品
- 実務ワークフロー向けの開発者ツール
- CLI、構造化出力、RPC 連携、macOS デスクトップ配布に対応

## 公開されている主な機能

- 対話型ターミナル利用
- 単発プロンプトと構造化出力
- ファイル、検索、shell の内蔵ツール
- セッション保存と再開
- Provider 設定
- skills、prompts、themes、基本拡張ランタイム

## なぜこの公開構成なのか

公開リポジトリは主に次を担うべきです。

- ランタイムコード
- インストールとパッケージング用スクリプト
- ユーザー向けドキュメント
- セキュリティ文書
- Provider と連携のガイド
- リリース履歴

内部向け戦略メモや一時的な計画資料を、製品の標準入口にすべきではありません。
