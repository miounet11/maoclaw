# サポート範囲

このページは `猫爪 v0.1` の日本語サポート範囲要約です。

## v0.1 で主にサポートするもの

- macOS
- 対話型 CLI
- print / JSON 出力
- 基本 RPC
- Anthropic、OpenAI、Gemini、Azure OpenAI
- `read`、`write`、`edit`、`bash`、`grep`、`find`、`ls`
- セッション保存、継続、再開、分岐、圧縮

## v0.1 で重点的に保証するもの

- インストールと初回起動の明確な導線
- 基本的な Provider 設定
- 主経路のツール利用
- コード理解と軽微な修正フロー

## v0.1 が約束しないもの

- すべての歴史的ツールチェーンや外部エコシステムの網羅
- すべてのプラットフォームで同等の成熟度
- すべての高度な連携ケースで完全安定
