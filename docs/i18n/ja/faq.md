# FAQ

## 猫爪 / maoclaw とは何ですか？

中国で独立開発された AI コーディングエージェントプロジェクトで、現在はターミナル中心の Rust ランタイムを核にしています。

## なぜコマンドはまだ `pi` なのですか？

移行コストを下げるため、現段階では互換コマンドを維持しています。意図的な仕様です。

## もう完全に安定した商用版ですか？

まだです。公開姿勢は試用版と継続改善です。

## いま広範囲に安定した正式版ですか？

まだです。現在の公開姿勢は、焦点を絞った試用版と継続改善です。

## いま何に向いていますか？

- ローカルリポジトリ解析
- コード検索と説明
- 軽量な修正提案
- セッション継続
- 基本的な RPC 連携

## 現在の内蔵ツールは？

- `read`
- `write`
- `edit`
- `bash`
- `grep`
- `find`
- `ls`

## 最初に選ぶ Provider は？

- Anthropic
- OpenAI
- Google / Gemini
- Azure OpenAI

## 初回失敗時に何を確認すべきですか？

1. `pi --version`
2. API キーがあるか
3. 現在のディレクトリが正しいか
4. Provider が利用可能か
5. まず最も簡単なコマンドから試したか
